# FFmpegFreeUI 页面布局复刻文档

## 一、主窗口结构

### 1.1 整体布局

```
┌──────────────────────────────────────────────────────────────────┐
│  窗口标题栏: FFmpegFreeUI {版本号}                                 │
├────────┬─────────────────────────────────────────────────────────┤
│        │                                                          │
│  左侧  │                    主内容区域                            │
│  导航  │                                                          │
│  选项卡│              (根据选中的选项卡显示不同内容)               │
│        │                                                          │
│        │                                                          │
│        │                                                          │
└────────┴─────────────────────────────────────────────────────────┘
```

### 1.2 窗口尺寸
- 最小尺寸: 1200 x 700 像素
- 默认尺寸: 1200 x 700 像素
- 支持高 DPI 缩放

### 1.3 配色方案
- 主背景色: `#181818` (RGB: 24, 24, 24)
- 次背景色: `#303030` (RGB: 48, 48, 48)
- 面板背景色: `#242424` (RGB: 36, 36, 36)
- 主文字色: `Silver` / `#C0C0C0`
- 次文字色: `Gray` / `#808080`
- 强调色: `YellowGreen` (黄绿色)


**任务列表列定义:**
1. 状态图标
2. 文件名
3. 进度
4. 文件大小
5. 编码参数
6. 输出路径
7. 其他信息

**操作逻辑:**
- 拖拽文件到列表: 添加任务
- 双击任务: 打开参数编辑
- Enter: 开始任务
- Space: 暂停任务
- Delete: 移除任务
- Ctrl+A: 全选
- Alt+A: 反选

## 四、操作流程

### 4.1 标准使用流程

```
1. 启动软件 → 显示起始页
       ↓
2. 添加文件 → 编码队列页面 (拖拽或点击添加)
       ↓
3. 设置参数 → 参数面板页面 (选择编码器、质量、滤镜等)
       ↓
4. 开始编码 → 编码队列页面 (点击开始按钮)
       ↓
5. 监控进度 → 编码队列页面 (查看进度、状态)
       ↓
6. 完成 → 查看输出文件
```

### 4.2 快捷键

| 快捷键 | 功能 |
|--------|------|
| Enter | 开始任务 |
| Space | 暂停任务 |
| Delete | 移除任务 |
| Ctrl+A | 全选任务 |
| Alt+A | 反选任务 |
| Escape | 重置任务 |
| End | 停止任务 |
| F3 | 上移任务 |
| F4 | 下移任务 |

### 4.3 拖拽行为

- 拖拽文件到队列: 使用当前参数方案添加任务
- 拖拽文件 + Ctrl/Shift/Alt: 为文件使用单独参数方案

## 五、ffmpeg_desktop 复刻建议

### 5.1 当前实现 vs 原版对比

| 功能模块 | ffmpeg_desktop | FFmpegFreeUI | 差距 |
|---------|---------------|--------------|------|
| 左侧导航 | 无 | 13个选项卡 | 需要添加 |
| 编码队列 | 有（简化版） | 完整 | 需要增强 |
| 参数面板 | 分散在各组件 | 集中式选项卡 | 需要重构 |
| 任务管理 | 基本功能 | 完整 | 需要增强 |
| 输出面板 | 无 | 有 | 需要添加 |
| 预设管理 | 有 | 有 | 基本一致 |

### 5.2 推荐重构方案

1. **添加左侧导航选项卡**
   - 使用 Element Plus 的 Tabs 组件
   - 垂直布局，左对齐

2. **重构主页面布局**
   - 将当前的参数面板改为选项卡式布局
   - 将各功能面板整合到参数面板的子选项卡中

3. **增强编码队列页面**
   - 添加状态栏显示全局信息
   - 添加工具栏按钮
   - 添加输出面板（可折叠）

4. **统一参数面板**
   - 将 TrimPanel、FilterPanel、SubtitlePanel、ColorPanel、StreamPanel 整合为参数面板的子选项卡
   - 添加参数总览选项卡

### 5.3 新增页面建议

| 页面 | 优先级 | 说明 |
|------|--------|------|
| 起始页 | 中 | 欢迎信息、快速入门 |
| 准备文件 | 低 | 批量添加功能 |
| 媒体信息 | 中 | 查看文件详情 |
| 设置页 | 高 | 软件配置 |
